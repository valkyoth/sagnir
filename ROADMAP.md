# Sagnir Roadmap

Status: planning document

Sagnir reaches 1.0.0 when the `saga` CLI is serious enough for local-first
source-state work: initialize a realm, inspect worktree state, create worlds,
seal changes, verify proofs, promote state between worlds, bundle state for
offline transfer, and sync through the Sagnir protocol.

The detailed release plan lives in [Version Plan](docs/VERSION_PLAN.md).
